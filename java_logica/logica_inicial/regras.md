# Lógica de Programação com Java

Nesta pasta estudamos lógica de programação usando a linguagem Java, evoluindo progressivamente pelos seguintes tópicos:

## 1. Estrutura Sequencial

Programas que executam linha a linha, sem desvios. Foco em entrada de dados, processamento e saída.

**Conceitos praticados:**

- Declaração de variáveis (`int`, `double`, `String`)
- Leitura de dados com `Scanner`
- Operações aritméticas
- Saída formatada com `System.out.println` e `System.out.printf`

### Exercícios

| Arquivo | Descrição |

---

## 2. Estrutura de Decisão

Programas que tomam decisões com base em condições.

**Conceitos a praticar:**

- `if / else if / else`
- `switch / case`
- Operadores relacionais (`>`, `<`, `==`, `!=`) e lógicos (`&&`, `||`, `!`)

---

## 3. Estrutura de Repetição

Programas que repetem blocos de código.

**Conceitos a praticar:**

- `for`
- `while`
- `do-while`
- `break` e `continue`

---

## 4. Funções (Métodos)

Organização do código em blocos reutilizáveis.

**Conceitos a praticar:**

- Declaração de métodos com e sem retorno
- Parâmetros e argumentos
- Escopo de variáveis
- Sobrecarga de métodos

---

## 5. POO — Programação Orientada a Objetos

Modelagem de sistemas usando classes e objetos.

**Conceitos a praticar:**

- Classes e objetos
- Atributos e métodos
- Construtores
- Encapsulamento (`private`, `public`, getters e setters)

### 5.1 Herança

- Reutilização de código entre classes (`extends`)
- Sobrescrita de métodos (`@Override`)
- Palavra-chave `super`

### 5.2 Polimorfismo

- Polimorfismo em tempo de execução
- Interfaces e classes abstratas

---

## 6. Construção de APIs

Criação de APIs REST com Java.

**Conceitos a praticar:**

- Spring Boot (setup e estrutura)
- Controllers, Services e Repositories
- Verbos HTTP: `GET`, `POST`, `PUT`, `DELETE`
- Retorno de JSON

---

## Padrão dos Arquivos

- Um exercício por arquivo `.java`
- Nome do arquivo: `exercicio_N.java`
- Package: `logica_inicial`
- Classe pública com o mesmo nome do arquivo
